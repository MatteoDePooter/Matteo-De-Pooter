De Opstart.

01. Wat heb ik geleerd?

Beschrijf hier kort wat je deze week hebt gedaan. Bijvoorbeeld: de installatie van Node.js, het begrijpen van de mappenstructuur en hoe Tailwind CSS samenwerkt met je HTML-bestanden.

02. Problemen & Oplossingen

Ik liep vast bij het uitvoeren van scripts in PowerShell (Execution Policy).

Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned

Door dit commando te gebruiken, kon ik de NPM scripts veilig uitvoeren zonder dat Windows de toegang blokkeerde.

03. Reflectie

In het begin voelde de workflow met de terminal een beetje intimiderend, maar nu ik zie hoe Tailwind de styling automatiseert, begrijp ik de kracht van deze tools. Volgende week wil ik dieper ingaan op de grid-layouts.
